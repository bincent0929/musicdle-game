How To Play

Here's some info about the game before you start.

🎧 More song with every guess: You start with 6 seconds of audio. Every guess grants +6 seconds for you to guess the song. (max 30s).

🎯 Smarter guesses: Type an artist’s name or the name of the song in the search bar, and you'll see a variety of choices.

🪄 If you just don't know it: Reveal shows the answer but you lose the game. You should only use it when you really aren't sure what it is.

Press the arrow to start playing. Refresh the page to get these tips again.
